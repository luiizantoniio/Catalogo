📤 Publicar Catálogo no GitHub

Exporta o arquivo produtos.json com todos os itens. Suba este arquivo no seu repositório GitHub para atualizar o catálogo público.

Como publicar:
1. Clique em Exportar produtos.json
2. Acesse seu repositório no GitHub
3. Clique no arquivo produtos.json → botão ✏ (editar) → arraste o novo arquivo
4. Clique em Commit changes
5. Aguarde ~1 minuto → catálogo atualizado! 🎉
